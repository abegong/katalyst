    Parent directory in

    • Check type ID
    • Purpose
    • Configuration keys
    • Example

    Check type ID#

    kind: filesystem_parent_dir_in

    Purpose#

    Require that the file’s parent directory name is in an allowed set.

    Configuration keys#

    FieldRequiredDefaultMeaning
    valuesyes-Allowed parent directory names.

    Example#

    collections:
  notes:
    path: notes
    checks:
      - kind: filesystem_parent_dir_in
        values: [books, people]
