⚠️ Katalyst is in its earliest days.

I'm actively building it in the open, which means things are incomplete, rough in places, and likely to change without notice.

APIs, commands, config formats, and concepts can all break between commits. Please don't rely on it for anything that needs to be stable yet.

I'd really appreciate your feedback as it takes shape. Please share it in a public thread: open an issue for bugs, questions, or requests, or open a pull request if you have a fix.
