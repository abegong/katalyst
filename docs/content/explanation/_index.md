# Explanation

Understanding-oriented discussion of the *why* behind Katalyst: the data
model it descends from, the domain it operates on, and the rationale behind
its design choices. These pages serve both users who want the bigger picture
and contributors who need the conceptual map. Decision rationale lives here,
distributed onto the topic it explains rather than in a central log.
