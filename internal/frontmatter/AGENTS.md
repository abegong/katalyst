# internal/frontmatter

Parses and formats markdown frontmatter, with source-line tracking, and owns
the canonical form `katalyst fix` rewrites files into.

Architecture and design rationale - the parsed document model, why `fix` is
deliberately opinionated, and why it never injects missing values - live in the
[Frontmatter and fix](../../docs/content/deep-dives/formatting.md) deep-dive,
which is the source of truth. This file only points to it.
